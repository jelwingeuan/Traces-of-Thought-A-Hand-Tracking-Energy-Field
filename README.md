# Gesture Bloom: Touchless Energy Field

Gesture Bloom is an interactive audiovisual installation created in TouchDesigner. The project explores touchless interaction by using webcam-based hand tracking to transform human gestures into glowing smoke-like digital energy.

The artwork allows users to interact without touching a keyboard, mouse, or screen. Instead, the user’s hand becomes the controller. As the hand moves, the visuals respond through light, colour, blur, feedback, and motion. The result is an abstract energy field that feels alive and reactive.

## Concept

This project focuses on the relationship between human gesture and digital presence. The hand is normally used to touch, hold, write, and communicate. In this artwork, the hand does not physically touch anything, but its movement still affects the digital environment.

The glowing trails represent invisible emotion, thought, and presence becoming visible in digital space. Slow movement creates a calm and soft visual response, while stronger movement creates brighter, more chaotic energy.

## Interaction

The interaction is based on hand tracking through a webcam.

- Hand movement controls the position and behaviour of the visual energy.
- Faster movement increases intensity and creates a more chaotic feeling.
- The glowing trails and feedback effects create a sense of digital residue left behind by the user’s gesture.

## Tools Used

- TouchDesigner
- Webcam input
- Hand tracking / computer vision
- CHOPs for interaction data
- TOPs for visual effects, feedback, blur, and colour processing

## Final Output

The final prototype is a real-time interactive audiovisual artwork designed to run in TouchDesigner Perform Mode. It demonstrates how natural body movement can be used as an interface for generative digital art.
